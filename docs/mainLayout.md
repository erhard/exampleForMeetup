## mainLayout

Die Datei mainLayout bildet sozusagen den Rahmen um die Applikation. Hier ist das Menü, login, Sprachauswahl etc.
Auch zentrale Komponenten werden hier eventgesteuert aufgerufen.
In dem mainLayout ist ein unscheinbarer Tag :  router-view  Dieser Tag wird durch den jeweiligen SeitenInhalt ersetzt, der durch den router ausgelöst wird. Der Router wiederum wird durch das Menü angesprochen.
